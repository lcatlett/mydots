# iTerm2 Color Themes

Color scheme files (`.itermcolors`) for iTerm2.

## Importing a Theme

1. Open iTerm2 → Settings → Profiles → Colors
2. Click **Color Presets…** → **Import…**
3. Select an `.itermcolors` file from this directory
4. Choose the imported preset from the dropdown

## Files

| File | Description |
|------|-------------|
| `lindsey.itermcolors` | Primary color scheme |
| `lindsey2.itermcolors` | Alternate variant |
| `profile2.itermcolors` | Secondary profile scheme |
| `another.itermcolors` | Additional theme |

## What's Not Tracked

The full iTerm2 profile plist (`com.googlecode.iterm2.plist`) and state exports (`.itermexport`) are gitignored — they contain personal session state and window arrangements that aren't portable.
